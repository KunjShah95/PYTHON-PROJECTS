# Password Generator Tool

This is a simple Python-based password generator tool that allows you to create strong and secure passwords based on your specified criteria.

## Features

*   **Customizable Length:** Specify the desired length of the password.
*   **Character Set Selection:** Choose which character types to include (uppercase, lowercase, digits, symbols).
*   **Random Password Generation:** Generates a random password based on your selections.
*   **User-Friendly Interface:** Easy to use and understand.

## How to Use

1.  **Clone the Repository:**
    ```
    git clone <repository_url>
    ```
2.  **Navigate to the Directory:**
    ```
    cd password-generator-tool
    ```
3.  **Run the Script:**
    ```
    python password_generator.py
    ```
4.  **Follow the Prompts:** The script will ask you to input the desired password length and character types.

## Requirements

*   Python 3.x
